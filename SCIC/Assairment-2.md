## **Task: GitHub Profile Task**

**Objective:** Create a well-structured and professional GitHub profile that effectively showcases your work, skills, and contributions.

**Task Requirements:**

1. Profile Picture

- Choose a profile picture that is neither too formal nor too casual. Maintain a professional yet approachable look. (**For female students, use of logo is allowed if there is any issue in sharing own picture.**)

**2. Location and Email Address**

- Add your current location.
- Include your professional email address, contact number for contact purposes.

**3. Profile README**

Your GitHub profile must contain a well-designed README.md file with the following elements:

- Banner Image: A visually appealing banner image at the top.
- Name & Designation: Clearly mention your full name and professional designation (e.g., Frontend Developer, Web Developer, Full Stack Developer, etc.).
- About Me Section: Write a short introduction about yourself in 3-4 lines.
- Current Activities: Include a bullet-point list of your latest activities, such as:
- I am exploring Next.js.
- I’m working on a tourism website.
- Skills Section:
- Use icons to visually represent your skills instead of just bullet points.
- Categorize skills if necessary (e.g., Frontend, Backend, DevOps, Tools).
- Social Links:
- Add relevant social media and professional links (e.g., GitHub, LinkedIn, Twitter).
- Do not include Instagram, Snapchat, TikTok, etc.
- Relevant GitHub Stats: Display contribution statistics like:
- Example:
- Number of contributions
- Languages used
- Streak data

**4. Pinned Repositories (At Least 3 client-side Repositories)**

- Pin at least three repositories that best showcase your work. (You can use course assignments/other projects.)

**5. Repository Details (For Each Pinned Repository)**

Each repository must include the following:

- Description:
- A short project overview.
- Live project link.
- Technologies used.
- README File:
- A concise project overview.
- Add a clean screenshot (if possible).
- List the main technologies used.
- Highlight the core features of the project.
- Mention the dependencies used.
- Provide a step-by-step guide on how to run the project locally.
- Include live project links and other relevant resources.

## **Task: LinkedIn Profile Task**

1. **Profile Basics [Follow the following 4 points]**

- **Profile Photo:** Upload a high-quality, professional picture . Ensure the background is clean.
- **Background Photo:** Add a relevant image, such as a sleek tech-themed graphic or a personal logo.
- **Headline:** Write a compelling headline that reflects your expertise and what you do.
- **Custom URL:** Edit your LinkedIn URL to make it clean:
- **Example:** linkedin.com/in/yourname

**2. About Section [Follow the following 3 points]**

- Write a concise summary highlighting the following 3 points:
- Your technical expertise
- Career achievements
- Passion and goals
- Add contact information.
- Use bullet points for readability.

**3. Experience Section ( If you Have Any) [Follow the following 2 points]**

- List relevant roles with detailed descriptions containing the following 4 points.
- **Title:** Position which you served,
- Company/organization name,
- Start and end date.
- **Description having the following 3 points.**
- Showcase your contribution in the organization.
- Mention specific projects and technologies used.
- Explain your detailed responsibilities.
- Include freelance or open-source projects if applicable containing all the above points.

**4. Skills**

- Add at least 12 key skills containing all the 3 following points.
- Programming languages (e.g., JavaScript, Python).
- Frameworks/libraries (e.g., React, Angular).
- Tools/technologies (e.g., Git, Docker, AWS).

**5. Projects Section[Follow the following point]**

- Showcase at least 2 best projects with each containing the following 3 points.
- Provide project names, brief descriptions including core services of the website and live link.
- skills you used.
- Screenshot of home page in media part.

**6. Certifications ( If you have any)**

- Add relevant certifications, such as:
- FreeCodeCamp
- Coursera (e.g., Web Development with React)
- Google’s UX Design Certificate
- AWS Certified Developer
- Include name, issuing organizations, issue date and expiration date.

**7. Education ( only focus on your last education ) [Please focus on the following point]**

- Mention academic background, focusing on relevant fields:
- Institution/School, Degree, Start date and end date.
- Add relevant coursework or projects completed during studies in the description part.

**8. Language [Please focus on the following point.]**

- Add all the languages you know.
- Example: Bangla, English, Hindi ,etc.
